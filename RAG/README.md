# Hybrid RAG Core Logic

Folder ini berisi logika inti (core engine) dari sistem RAG. 

**⚠️ PENTING:** Untuk menjalankan aplikasi, sangat disarankan menggunakan **`cli.py`** yang ada di direktori utama (root) karena sudah menangani pengaturan path dan konfigurasi secara otomatis.

### Penggunaan yang Direkomendasikan (via Root):
1.  **Ingest**: `python3 cli.py ingest`
2.  **Chat**: `python3 cli.py chat`
3.  **API**: `python3 cli.py serve`

Silakan merujuk pada **`README.md` di direktori utama** untuk panduan instalasi dan penggunaan lengkap.
